## About
This project is a fullstack app with authorization system. The application allows you to create employees. An employee can only be edited or deleted by its creator (user).

## Used Technologies

- ReactJS
- Redux Tookit
- Typescript
- RTK Query
- Antd
- Prisma client
- Jsonwebtoken
